# Contributing

欢迎提交改进。

适合贡献的内容：

- 新的网页 AI 接入说明
- DevSpace / MCP SuperAssistant 排错案例
- 固定域名和隧道配置示例
- 更安全的脚本模板
- Windows / macOS / Linux 差异说明

提交前请确认：

- 不包含真实 token、cookie、账号、域名、临时隧道地址
- 不包含本机绝对路径或私人目录
- 不包含项目私有数据
- 示例路径使用占位符

建议提交前运行：

```powershell
rg "Users|AppData|trycloudflare|ngrok-free|TOKEN|authtoken|secret|password"
```
