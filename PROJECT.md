# PROJECT: web_agents

`web_agents` 的长期方向是面向开发者和普通用户的双层 AI 工作台。

当前阶段优先建设浏览器插件端：

- 普通用户通过插件快速连接本地 MCP、管理权限、把任务插入网页 AI。
- 进阶用户可以手动选择多个网页模型进行任务分发和结果对比。
- 开发者可以扩展站点适配器、MCP 能力、权限网关和后续 Agent 工作流。

后续阶段再扩展为 GUI Studio：

- 可视化无代码 Agent 编排。
- 多网页账号绑定。
- 多模型协作会话。
- Claude Code、Codex、Kimi 等 Agent runtime adapter。
